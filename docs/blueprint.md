# **App Name**: Scrollyfolio

## Core Features:

- Hero Section: Full-height landing with animated name and title.
- Pinned Scrollytelling: Sticky container with synced text and image changes.
- Content Blocks: Vertical text blocks describing projects.
- Image/Video Column: Right column displaying visuals corresponding to text blocks.
- GSAP Animations: Smooth animations and ScrollTrigger integration.
- Footer Section: Simple footer with contact info and CV link.

## Style Guidelines:

- Background: Absolute black (#000000) to provide a stark canvas.
- Text: Pure white (#FFFFFF) for high contrast and readability.
- Font: 'Inter', sans-serif, known for its legibility and modern appearance, perfect for body and headline text.
- Text style: 'Tracking-tight', 'Leading-tight', for a Swiss Design aesthetic, making it precise and clean.
- Two-column layout: Left for scrollable text, right for fixed visuals.
- GSAP ScrollTrigger: Used to crossfade images in sync with scrolling text.
